Ce plugin permet de visionner un flux vidéo d'une caméra IP Vivotek et de contrôler quelques fonctions

Le plugin n'est qu'a son début et demande qu'à évoluer.

Vous devez installer vlc et firefox sur le serveur ou se trouve S.A.R.A.H.


1) Fonction :

Permet d'afficher un flux vidéo selon votre choix, et de contrôler quelques fonctionnalités de votre caméra


2) Paramétrage :

Dans les paramètres du plugin, vous devez renseigner les informations par vos valeurs :

  • adresse_ip : Adresse IP de votre caméra.
  • protocol : protocol utilisé pour vos flux vidéo.
    Ex : pour les flux H.264, utilisé le protocol rtsp, pour les flux mjpg ou encore jpg
  • num_port : Port de votre caméra.
  • nom_flux : Nom du flux vidéo.
    Ex : live.sdp / stream.mjpg / video.jpg
  • lien_capture : URL de la photo.
    Ex : "cgi-bin/viewer/video.jpg"
  • user : Nom d'un utilisateur d'un groupe "administrator" de la caméra. Par défaut l'utilisateur est "root".
  • password : Mot de passe du compte utilisateur


3) Les commandes:
  • Sarah, affiche la caméra.
  • Sarah, redemarre la caméra.
  • Sarah, reset la caméra.
  • Sarah, réinitialise la caméra.
  • Sarah, prends une photo avec la caméra.
