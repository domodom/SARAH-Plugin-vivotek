1) Fonction :

Permet de visionner et contôler une caméra IP Vivotek


2) Paramétrage :

vlc et firefox doivent être installer sur le serveur ou se trouve S.A.R.A.H.

Dans les paramètres du plugin, renseigner les informations par vos valeurs :

  • adresse_ip : Adresse IP de votre caméra.
  • protocol : protocol utilisé pour votre flux vidéo.
    Ex : pour les flux H.264, utilisé le protocol rtsp, pour un flux mjpg ou encore jpgutilisé le protocol http
  • nom_flux : Nom du flux vidéo.
    Ex : live.sdp / stream.mjpg / video.jpg
  • lien_capture : URL de la photo.
    Ex : "cgi-bin/viewer/video.jpg"
  • user : Nom d'un utilisateur d'un groupe "administrator" de la caméra. Par défaut l'utilisateur est "root".
  • password : Mot de passe du compte utilisateur


3) Les commandes:
  • Sarah, affiche la caméra. / SARAH, fermes l'affichage de la caméra.
  • Sarah, prends une photo avec la caméra. / SARAH, ferme la photo de la caméra.
  • Sarah, redémarre la caméra.
  • Sarah, passe la caméra en mode jour/nuit/auto.
  • Sarah, démarre la ronde de la caméra. / Sarah, stop la ronde de la caméra.
  • Sarah, active le masque de la caméra. / Sarah, désactive le masque de la caméra.
  • Sarah, positionnes la caméra sur [preset]
    Editer le fichier vivotek.xml pour renseigner vos presets
  • Sarah, Va a [droite - gauche - haut - bas]
  • Sarah, Zoom [+/-]
