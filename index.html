Permet de visionner et contôler une caméra IP Vivotek






Paramétrage :

vlc doit être installé sur le serveur ou se trouve S.A.R.A.H.

Dans les paramètres du plugin, renseigner les informations :

  • adresse_ip : 192.168.x.x
  • protocol : protocol utilisé par le flux vidéo.
    Pour les flux H.264, utilisé le protocol rtsp, pour un flux mjpg ou encore jpg utilisé le protocol : http
  • nom_flux : Nom du flux vidéo.
    /live.sdp - /stream.mjpg
  • lien_capture : URL de la photo.
    /cgi-bin/viewer/video.jpg
  • user : utilisateur de la caméra. Par défaut l'utilisateur est "root".
  • password : Mot de passe du compte
  • systeme : windows - linux


Les commandes:
  • Sarah, affiche la caméra. / SARAH, fermes l'affichage de la caméra.
  • Sarah, prends une photo avec la caméra.
  • Sarah, active/désactive l'alarme de la caméra.
  • Sarah, redémarre la caméra.
  • Sarah, passe la caméra en mode jour/nuit/auto.
  • Sarah, démarre/stop la ronde de la caméra.
  • Sarah, active/désactive le masque de la caméra.
  • Sarah, positionnes la caméra sur [preset]
    Editer le fichier vivotek.xml pour renseigner vos presets
  • Sarah, Va a [droite - gauche - haut - bas]
  • Sarah, Zoom [+/-]
